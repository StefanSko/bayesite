# Changelog

All notable user-visible changes to Bayesite are recorded here.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).
Bayesite uses semantic versioning while its public formats remain explicitly
versioned on their own.

## [Unreleased]

### Added

- Add a mandatory, stdlib-only SBC rank-uniformity conformance gate using
  Monte Carlo-calibrated simultaneous ECDF confidence bands.

## [0.2.1] - 2026-07-10

### Added

- Decode, bind, transform, evaluate, and forward-simulate Bayeswire
  `VectorBounds` constraints for censored and partially observed vectors.
- Validate the final Bayeswire corpus, including closed namespaced `Submodel`
  declarations flattened to ordinary dotted IR names.

### Changed

- Resolve finite base-distribution support for `VectorBounds` from the unique
  same-name direct or scatter owner, independent of stochastic-site order.
- Classify declaration-backed sites before prior-predictive drawing and reject
  additional density Factors that have no supported ancestral semantics.
- Refresh the vendored Bayeswire specification and corpus to merged bayescycle
  commit `e9784f5`.

### Fixed

- Preserve stable bounded draws in extreme distribution tails.
- Align vector support through wrapped negative missing indexes and validate
  missing-value shape against `missing_idx` before drawing.
- Defer support folding when a required support value is generated by an
  earlier stochastic site.

## [0.2.0] - 2026-07-06

### Added

- Support the canonical wrapped data document and neutral model/data
  fingerprint contract.
- Add the machine-readable `capabilities` command.
- Implement explicit `Truncated` distributions and stable far-tail simulation.
- Add merge-blocking validation, cross-backend conformance, and a pinned
  nuts-rs statistical oracle.

### Changed

- Adopt the neutral `bayeswire_ir` envelope and vendored Bayeswire
  specification/corpus snapshots.

## [0.1.0] - 2026-06-24

### Added

- Initial zero-dependency Bayesite CLI and Rust runtime for decoding Bayeswire
  IR, evaluating log densities, running NUTS, diagnostics, predictive
  workflows, recovery, and SBC.
- Reproducible release archives and checksums for Linux, macOS, and Windows.
